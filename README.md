## Installation

Install Python in your Workspace

Python 3.9.12

```bash
 https://www.python.org/downloads/release/python-3912/
```
Install Django

```bash
 pip install Django
```
Command to run my project locally

```bash
 python manage.py runserver
```
Open the link in your browser
```bash
 http://127.0.0.1:8000/
```
